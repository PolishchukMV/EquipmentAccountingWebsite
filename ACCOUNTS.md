# Учётные записи системы учёта оборудования

## Администратор (суперпользователь)

| Поле | Значение |
|------|----------|
| **URL входа** | http://localhost:8000/admin/ |
| **Имя пользователя** | `admin` |
| **Пароль** | `admin123` |
| **Email** | `admin@example.com` |
| **Имя** | Администратор |
| **Фамилия** | Системы |
| **Роль** | Суперпользователь (полный доступ) |

### Создание суперпользователя

Если суперпользователь не был создан, выполните:

```bash
python create_superuser.py
```

Или вручную через Django:

```bash
python manage.py createsuperuser
```

---

## Другие роли пользователей

Система поддерживает 4 роли пользователей:

### 1. Admin (Администратор)
- Полный доступ ко всем функциям
- Управление пользователями
- Доступ к админ-панели

### 2. Manager (Менеджер)
- Создание/редактирование/удаление оборудования
- Управление подразделениями
- Обслуживание оборудования

### 3. Employee (Сотрудник)
- Просмотр оборудования
- Просмотр своей ответственности
- Обратная связь

### 4. Auditor (Аудитор)
- Просмотр отчётов
- Просмотр истории изменений
- Аудит системы

## Регистрация пользователей

Обычные пользователи могут регистрироваться через:
- URL: `/accounts/register/`
- Или через админ-панель

## Безопасность

⚠️ **ВАЖНО:** После первого входа измените пароль администратора!

Рекомендации:
- Используйте сложные пароли (минимум 12 символов)
- Не используйте один пароль для нескольких систем
- Регулярно меняйте пароли
- Включите двухфакторную аутентификацию (если доступна)

## Скрипты управления

### Создание суперпользователя
```bash
python create_superuser.py
```

### Просмотр всех пользователей
```bash
python -c "import os, django; os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'equipment_tracking.settings'); django.setup(); from django.contrib.auth import get_user_model; User = get_user_model(); [print(f'{u.username} - {u.email} - Superuser: {u.is_superuser}') for u in User.objects.all()]"
```

### Сброс пароля
```bash
python manage.py changepassword admin
```
