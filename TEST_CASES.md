## Модуль: Accounts (Учёт пользователей)

### TC-ACC-001: Регистрация нового пользователя

**ID:** TC-ACC-001  
**Приоритет:** Высокий  
**Тип:** Позитивное функциональное тестирование

**PreConditions:**
- Пользователь не авторизован
- Открыта страница регистрации

**Steps:**
1. Перейти на страницу `/accounts/register/`
2. Ввести имя пользователя: `testuser`
3. Ввести email: `test@example.com`
4. Ввести пароль: `TestPass123!`
5. Ввести подтверждение пароля: `TestPass123!`
6. Нажать кнопку "Зарегистрироваться"

**Expected Result:**
- Пользователь успешно создан
- Сообщение "Регистрация успешна"
- Автоматический переход на страницу входа
- Пользователь может войти с новыми данными

## Модуль: Equipment (Оборудование)

### TC-EQP-001: Создание оборудования с корректными данными

**ID:** TC-EQP-001  
**Приоритет:** Высокий  
**Тип:** Позитивное функциональное тестирование

**PreConditions:**
- Пользователь авторизован как Manager
- Существует категория оборудования

**Steps:**
1. Перейти на страницу списка оборудования
2. Нажать кнопку "Добавить"
3. Заполнить форму:
   - Инвентарный номер: `INV-001`
   - Название: `Ноутбук Dell Latitude`
   - Категория: `Электроника`
   - Производитель: `Dell`
   - Модель: `Latitude 5420`
   - Серийный номер: `SN123456`
   - Статус: `На складе`
   - Цена покупки: `50000`
4. Нажать "Сохранить"

**Expected Result:**
- Оборудование успешно создано
- Сообщение "Оборудование успешно создано"
- Перенаправление на список оборудования
- В журнале изменений создана запись
