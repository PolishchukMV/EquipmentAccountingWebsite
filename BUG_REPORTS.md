# Баг-репорты системы учёта оборудования

## Инструкция по заполнению

### Серьёзность (Severity)
- **S1 - Blocker:** Критическая ошибка, блокирующая работу системы
- **S2 - Critical:** Серьёзная ошибка, влияющая на ключевой функционал
- **S3 - Major:** Ошибка средней тяжести, есть обходной путь
- **S4 - Minor:** Небольшая ошибка, не влияющая на функционал
- **S5 - Trivial:** Косметическая проблема

### Приоритет (Priority)
- **High:** Исправить немедленно
- **Medium:** Исправить в текущем спринте
- **Low:** Исправить при возможности

### BR-001: Ошибка при создании оборудования с пустым инвентарным номером

| Поле | Значение |
|------|----------|
| **ID** | BR-001 |
| **Название** | Ошибка при создании оборудования с пустым инвентарным номером |
